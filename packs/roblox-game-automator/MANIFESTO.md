# 🏙️ Metropolis: The Autonomous City-State
### *An Open Model Contract implementation for level-scale procedural generation.*

This repository contains the **Bridge Infrastructure** and **Autonomous Swarm Code** for the **Chaos Metropolis** project—a Gothic-Cyber city manifestation built via human-AI collaboration.

---

## 🚀 The Architecture
The project utilizes a **Hybrid Loader Architecture** to split level generation from player mechanics:
*   **Server Swarm**: Procedural generation of **Obsidian Skyscrapers**, **Gothic Arches**, and **Neon Street Grids**.
*   **Client Swarm**: **Radiant Player Glow**, **Sonic Dashes (Zip)**, and **Hacker Pulse** visual effects.

## 🌉 Key Components
*   **Bridge Server (Node.js)**: The central escrow for AI-generated Luau modules.
*   **Hybrid Loader (Luau)**: A "Bulletproof" Bypass script for Roblox that executes code directly in memory without security restrictions.
*   **Metropolis Primer (JSON)**: The "Canonical Document" that defines the level's theme, palette, and lighting before manifestation.
